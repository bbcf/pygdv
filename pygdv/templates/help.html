Main view

This is the principal interface view. Here you can find all data you have uploaded. From there you can create new projects, add data (tracks) to them and start working/visualizing.

Circles

With circles you are able to quickly collaborate between people. These are group of persons. You can create and add some people to them. Add people to a circle and you will be able to share your projects with them. Enter the person email with the edit link. You have also possibility to remove people from your circles with the delete link.

Sharing
With GDV you have the possibility to share a project with someone else. You can do it :
  • Via circles : sharing with a group of people.
  • Via links : public and download links.
    • The public link is an access to the view of your project. User with that link will be able to see all tracks in your project, but not to download or modify the data in it.
    • The download link enable the entire copy of your project by others. It's a permanent access to the data of your project.
When you want to share a project with a circle the project will appears in the Shared project section on the main interface of all user that are in the circle.

You can give three different permissions to a circle:

  • Read : User is able to see the data in your project but will not be able to download/add data to it
  • Download : User will be able to see and download the data in your project.
  • Upload : User will be able to edit the project (name) including adding or deleting tracks.

Jobs

With GDV you have the possibility to start some computation from the interface. On the project view, you have possibility to choose from some Operations to apply on your tracks. Each operation will appears on the Data panel. It's where you can fetch back result.

Assemblies

If the assembly you want to use is missing, use contact. Specify in the mail which assembly you want to have and it will be added. Even if the assembly is not very usual, contact us. It can be added very easily.

API

You can access GDV with a simple programming interface. Based on HTTP requests, you can create projects, add tracks, and prepare view without the web interface. There is a python script that you could use for this purpose. It's part of the bbcf utilities.

Quick start:

  • wget https://raw.github.com/bbcf/bbcfutils/master/Python/pygdv.py (fetch the file from github)
  • easy_install clize (install a third-party library - perhaps you will need a sudo)
  • python pygdv --help

In order to use it, you will need your personal key that you can get here.
