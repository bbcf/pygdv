Upload file

We call genomic files Tracks. To upload a new one, select Tracks tab and use the form to upload your data. You can enter URL or simply select it from your filesystem.

Don't forget to choose the right assembly for your file. See how to add an assembly .

See the data uploaded

You will be able to see the track(s) you uploaded by making Projects. Create one form the Projects tab, and add the track(s) with the button.

Share data

You can share projects with the button. You have two possibilities :

  • You have two links with read and download permissions that you can give to people that will be able to see the project and download data associated to the project respectivly (download permission includes read one).
  • You can share with just a group of persons. Select a circle and add it. You have one more permissions : upload that will give to this circle of person to upload data on the same project.

Create a circle

Go on Circles tab, and select the person(s) you want to add in it.

Assemblies

If the assembly you want to use is missing, use contact. Specify in the mail which assembly you want to have and it will be added. If the assembly is not very usual, contact us. It can be added very easily.

Access with an API

You can access GDV with a simple programming interface. Based on HTTP request, you can create projects, add tracks, and prepare view without the web interface. The simplest if the data you generate is at the end of an analysis pipeline. For more information on it you should read the python file that resume the requests to send to GDV.

In order to use it, you will need a key that you can ask.
