Welcome to pygdv

The next Genome Browser

Most genomics datasets can be viewed as features along the genome. Our aim is to provide a user interaction layer on top of a dynamic genomics data viewer running inside the web browser: the Genome Data Viewer (GDV). It was build to be faster and more intuitive to browse. It was build using JBrowse (Skinner, 2009). Moreover, it incorporate a set of tools to explore and analyse your tracks.

  1. Share and store

    Upload your genomics files on GDV. They are processed and kept for you. You will be able to download them at any time.

    Create projects. They can be shared with specific permissions to facilitate the interactivity between people.

  2. Play

    Perform analyses directly from the interface. See what is important and create new hypothesis quickly.
